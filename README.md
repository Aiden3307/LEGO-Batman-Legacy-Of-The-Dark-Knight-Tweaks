# LEGO-Batman-Legacy-Of-The-Dark-Knight-Tweaks

**LEGO Batman: Legacy of the Dark Knights Tweaks** — это оптимизирующий мод, созданный специально для модификации *LEGO Batman: Legacy of the Dark Knights*. Цель проекта — повышение производительности и обеспечение более плавного игрового процесса.

## Основные изменения

* **Оптимизация FPS:** Значительно увеличен и стабилизирован общий показатель кадров в секунду.
* **Устранение задержек:** Минимизированы статтеры и микрофризы во время игрового процесса.
* **Оптимизация видеопамяти:** Снижен и оптимизирован расход VRAM, что разгружает видеокарту.

## Влияние на графику

Изменения в визуальной составляющей присутствуют, но они минимальны. Единственное заметное отличие — это отключение тумана, что практически не бросается в глаза во время игры и не портит общую атмосферу.

## Инструкция по установке

1. **Скачайте архив:** Загрузите архив с модификацией из репозитория.
2. **Перейдите по пути:** Откройте проводник Windows и перейдите в папку конфигурации, скопировав этот адрес:
   ```text %LOCALAPPDATA%\Dinner\Saved\Config\Windows
3. **Установите файл:** Из скачанного архива перенесите файл Engine.ini в открывшуюся папку (с заменой, если потребуется).
4. **Убедитесь что галочка только для чтения включена:**
   * Нажмите правой кнопкой мыши (ПКМ) по файлу Engine.ini и выберите пункт Свойства.
   * В окне свойств (на вкладке «Общие») поставьте галочку напротив пункта Только для чтения.
   * Нажмите Применить и затем ОК.

Наслаждайтесь игрой!
